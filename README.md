# Geotimize Dataset

Local business data for AI agents. Updated every 6 hours.

## Contents

| File | Description |
|------|-------------|
| `by-country/{CODE}.json` | Per-country JSON (sample if >95 MB) |
| `by-country/{CODE}.jsonl` | Per-country JSONL (sample if >95 MB) |

## Full Data

Complete global data and large country files are available on **Hugging Face**:

https://huggingface.co/datasets/geotimize/businesses

Includes: `data.json`, `data.jsonl`, full `by-country/*` with archives.
